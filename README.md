# 🚀 Assistive Games & Interactive Environments for Stress Management

A collection of AI-powered gesture-controlled games and creative environments developed during the **DRID Summer Internship 2025-26** under the project:

> **“E-sports and Stress Management for Differently Abled People”**

This project explores how **computer vision, hand tracking, and interactive environments** can create accessible, stress-relieving, and engaging experiences for differently-abled users.

---

# 🌟 Projects Included

## 🍉 1. Hand-Tracking Fruit Ninja

A gesture-controlled arcade game where users slice falling fruits using real-time hand movements.

### Features

* Real-time hand tracking with MediaPipe
* Pinch gesture slicing
* Random fruit & bomb spawning
* Dynamic scoring system
* Smooth canvas animations

---

## 🐍 2. Snake Fun (Gesture-Controlled Snake Game)

A modern version of the classic Snake game controlled entirely using hand gestures.

### Features

* Finger-controlled snake movement
* Collision & score system
* Smooth gesture tracking
* Adaptive gameplay speed
* Accessible interaction design

---

## 🎨 3. AirBrush (Virtual Whiteboard Painter)

A virtual drawing application that turns the user's finger into a digital paintbrush.

### Features

* Air drawing using hand gestures
* Brush color & size controls
* Eraser and clear canvas tools
* Real-time drawing feedback
* Stress-relieving creative interaction

---

## 🪄 4. HoloBoard (2D & 3D Air Drawing Suite)

A gesture-based drawing suite supporting both 2D and 3D air drawing.

### Features

* 2D virtual whiteboard
* 3D air drawing visualization
* Save & replay drawings
* Real-time gesture tracking
* Interactive creative environment

---

## 🧊 5. HoloSculpt (3D Shape Creator)

A browser-based 3D creative tool where users manipulate shapes using hand gestures.

### Features

* Create 3D shapes using pinch gestures
* Move, scale & delete objects
* Neon-themed interactive visuals
* Three.js powered rendering
* Gesture-based 3D interaction
